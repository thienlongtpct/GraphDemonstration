Резюме

Это мой проект, который использует Spring Framework для бэк-энда, HTML5, CSS, Javascript для фронт-энда, чтобы построить граф и выполнить алгоритм в нем.

В моем проекте есть 6 алгоритмов:

  • Поиск в глубину - DFS
  • Поиск в ширину - BFS
  • Алгоритм Прима
  • Алгоритм Краскала
  • Алгоритм Дейкстры
  • Алгоритм Форда-Беллмана
Особенности

В этом проекте, я испольвовал языки и библиотеки ниже:

  • Для бзк-энда:

    • Библиотеки: Spring Framework, стандартные библиотеки.
    • Язык: Java.

  • Для фронт-энда:

    • Библиотеки: Bootstrap, Bootstrap Select, MathJax, Font Awesome, JQuery, CodeMirror (для code-snippet), SigmaJS (для построения графа)
    • Язык: HTML, CSS, Javascript.

Инструкция

Создать граф

  1. Запишите ребры в code snippet по следующему правилу:

    • Написать слово - вершина.
    • Напишите два слова - ребро, соединяющее 2 письменные вершины.
    • Запишите три слова - ребро с весом (меткой), является последним словом.
    • Внимание, если в строке более 4 слов, мы рассматриваем только первые три слова.

    2. Помните, что вы не можете редактировать количество вершины (он автоматически изменяется из-за данных вашего графика).

  2. Нажать кнопку "Создать граф".

  3. Если бы вы хотите удалить граф, нажать кнопку "Создать граф".

Демонстрация алгоритма

  1. Выбрать алгоритм, который вы хотите.

  2. Выбрать исходную вершину.

  3. Нажать кнопку "Начать алгоритм".

    • Чтобы посмотреть предыдущий этап алгоритма, нажать "Предыдущий"
    • Чтобы посмотреть следующий этап алгоритма, нажать "Следующий"
    • Чтобы закончить алгоритм, нажать "Закончить алгоритм"

    Помните, что вы не можете изменить, удалить граф или исходную вершину во время работы алгоритма

Объяснение

DFS - Поиск в глубину

Ноты для демонтрации вершин:

  • Синяя: начальная вершина.
  • Зеленая: вершина, которая сейчас посещается.
  • Красная: вершина, которая уже был посещена.
BFS - Поиск в ширину

Ноты для демонтрации вершин:

  • Синяя: начальная вершина.
  • Зеленая: вершина, которая сейчас посещается.
  • Желтая: вершина, которая находится в очереди, но не был посещена.
  • Красная: вершина, которая уже был посещена.
