Мои контакты

Это мой проект по дисциплине Программирования, который использует Spring Framework для бэк-энда, HTML5, CSS, Javascript для фронт-энда, чтобы построить граф и выполнить алгоритм в нем.

Особенности

В этом проекте, я испольвовал языки и библиотеки ниже.

Бзк-энд
  • Библиотеки: Spring Framework, стандартные библиотеки.

  • Язык: Java.
Фронт-энд
  • Библиотеки: Bootstrap, Bootstrap Select, MathJax, Font Awesome, JQuery, CodeMirror (для code-snippet), SigmaJS (для построения графа)
  • Язык: HTML, CSS, Javascript.

Использованные библиотеки для демонстрации

В этом проекте, я испольвовал библиотеки ниже для демонстрации.

SigmaJS

Sigma - это библиотека JavaScript, предназначенная для рисования графиков. Это облегчает публикацию сетей на веб-страницах и позволяет разработчикам интегрировать исследование сети в многофункциональные веб-приложения.

Sigma предоставляет множество встроенных функций, таких как рендеры Canvas и WebGL или поддержка мыши и сенсорного ввода, что делает работу с сетью на веб-страницах гладкой и быстрой для пользователя.

CodeMirror

CodeMirror - это универсальный текстовый редактор, реализованный на JavaScript для браузера. Он специализируется на редактировании кода и поставляется с рядом языковых режимов и дополнений, которые реализуют более продвинутые функциональные возможности редактирования.

Богатый API-интерфейс и система тем CSS доступны для настройки CodeMirror в соответствии с вашим приложением и расширения его новыми функциями.
